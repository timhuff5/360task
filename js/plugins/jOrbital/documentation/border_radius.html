Circles

CSS3 gives us a possibility to use border-radius. With a little abusement we can create nice oval bodies.

It's also useful for orbits debuging

The tricky thing is to set border-radius larger than element's edge.

Of course it's not working on IE :)

.orbital { 
	background:#fec; text-align:center;
	width:32px; height:32px; line-height:32px;
	border:2px solid #f88; 
	border-radius:64px; 
	-moz-border-radius:64px; 
	-webkit-border-radius:64px; }

.orbit { 
	border:2px solid #aaa; 
	border-radius:500px; 
	-moz-border-radius:500px; 
	-webkit-border-radius:500px; }

.orbit div { 
	background:#cef; text-align:center;
	width:32px; height:32px; line-height:32px;
	border:2px solid #88f; 
	border-radius:64px; 
	-moz-border-radius:64px; 
	-webkit-border-radius:64px; }
