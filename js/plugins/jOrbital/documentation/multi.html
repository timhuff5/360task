Multi dimensional orbits

Note that this funcionality is quite unstable and can provide you some unwanted results

Once again we include jQuery and jOrbital scripts.

/* jquery */
<script type="text/javascript" src="http://code.jquery.com/jquery-1.4.2.min.js"></script>
/* jquery.jOrbital plugin */
<script type="text/javascript" src="jquery.jorbital.js"></script>

<!-- let's have some basic CSS --> 
<style>
	.orbital { display:inline-block; width:32px; height:24px; margin:10px; 
		background:#a00; color:#fff; padding:4px; }
	.orbit > div { width:32px; height:32px; background:#4ac; color:#fff; 
		text-align:center; display: block; }
	.orbit .orbit > div { background:#8c4 }
</style>

<div class="orbital">
	Hover
	<div class="orbit">
		<div>1</div>
		<div class="orbital-2">2
			<div class="orbit">
				<div>1</div>
				<div>2</div>
				<div>3</div>
				<div>4</div>
				<div>5</div>
			</div>
		</div>
		<div>4</div>
		<div>5</div>
		<div>6</div>
	</div>
</div>

Of course you don't have to use DIVs. In fact you can use any TAG that suits you. Only thing to rememeber is structure and .orbit class

Now, lets make the orbits:

$(".orbital-2").jOrbital();
$(".orbital").jOrbital();

It's important to keep the right order of making orbitals. The deepest orbits should be created first. That's why we used two separated classes .orbital and .orbital-2

If everything is okay you should end-up with something like:
Hover
1
2
1
