Getting started

First of all, include jQuery and jOrbital scripts in document head section.

/* jquery */
<script type="text/javascript" src="http://code.jquery.com/jquery-1.4.2.min.js"></script>
/* jquery.jOrbital plugin */
<script type="text/javascript" src="jquery.jorbital.js"></script>

The second step is to create HTML for representing Orbital object, consisting Orbit with Planets.

<!-- let's have some basic CSS -->

<head> 
	<style>
		.orbital { outline:1px solid #f00; display:inline-block }
		.orbit { outline:1px solid #00f; }
	</style>
</head>

<div class="orbital">
	Hover
	<div class="orbit">
		<div>planet1</div>
		<div>planet2</div>
		<div>planet3</div>
		<div>planet4</div>
		<div>planet5</div>		
	</div>
</div>

I think the structure is quite obvious. Take a note that orbit class is mandatory.

Now put this javascript in your head section

<script type="text/javascript">
	// jQuery on document ready
	$(function() {
		$(".orbital").jOrbital();
	});
</script>

Yeah... it's that simple

If everything is okay you should end-up with something like:
Hover
planet1
planet2
planet3
planet4
planet5

Some important notes

I've given inline-block display to the orbital object, so it's not 100% wide.
